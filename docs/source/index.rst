Welcome to Documentation for DOLPHOT's JWST NIRCAM and NIRISS modules
===================================

**DOLPHOT** is a widely used software package for resolved stellar photometry.  It is written and mantained by Andrew Dolphin.  The main `DOLPHOT webpage <http://americano.dolphinsim.com/dolphot/>`_ hosts all necessary DOLPHOT files and the DOLPHOT user manuals.  This page provides information and examples for DOLPHOT applied to JWST imaging taken as part of the `Resolved Stellar Populations Early Release Science Program <https://ers-stars.github.io>`_.



Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
